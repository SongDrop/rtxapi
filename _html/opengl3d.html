Technical Explanation

This setup enables a full stereoscopic 3D gaming experience using the Xash3D-fwsg-OpenGL3D engine, which is an open-source fork of the Half-Life GoldSrc engine with stereoscopic 3D capabilities. The system supports:

  • Anaglyph 3D: Red-cyan glasses for basic stereoscopy on any screen.
  • Passive 3D: Side-by-side output for LG OLED TVs using polarized glasses, delivering full-color 3D without batteries.
  • Active Shutter / Quad-Buffer Stereo: Full-resolution 3D using compatible monitors and active shutter glasses, with a stereo OpenGL context.

The setup also integrates motion control using Nintendo Joy-Cons to replicate Wii-style free-mouse aiming. This allows players to aim using gyroscopic motion within a virtual rectangle, optionally scrolling the camera when reaching the edge, fully mimicking the Wii pointer experience.

Combining this engine, motion controls, and LG OLED TVs enables developers and players to experience real-time stereoscopic 3D gameplay with immersive cutscenes, interactive motion input, and full-color 3D output â€” creating a true cinematic gaming experience.
