Boot Method: UEFI and Legacy BIOS compatible

Use vhdusbdownloader (Electron app) to download the VHD from a URL directly to a USB drive and make it bootable. Large files (e.g., 256GB or more) can take many hours — for example, at 10MB/s a 256GB image takes roughly 7 hours to complete. The app includes built-in resume support, so if the transfer is interrupted you don’t need to start over.
