DISPOSABLE DEVELOPMENT ENVIRONMENT

Concept Overview

This system turns a VM into a disposable, fully-prepared development environment that is automated and collaborative. You can spin up multiple Unreal Engine projects, sync code, connect to OpenVPN, and trigger builds without manual intervention. Each VM is project-agnostic, allowing you to clone, test, and delete instances efficiently.

How It Works

1️⃣ VM Setup for Multiple UE Projects

  • Start from a base UE3 VM image.
  • On startup, a PowerShell script automatically:
    • Connects to OpenVPN
    • Checks if project repositories exist and pulls updates
    • Clones missing repositories
    • Opens projects in UE3 automatically
  • Result: same base image can run multiple projects instantly.

2️⃣ Git Integration + Collaboration

  • VS2022 runs locally while paired with VS Code Online or browser-based IDEs.
  • Developers can collaborate in real-time using GitHub/Forgejo.
  • VM startup scripts detect new commits and pull them into local VS2022 environment.
  • Optional webhooks can trigger builds automatically when updates occur.

3️⃣ OpenVPN + Incredibuild Automation

  • OpenVPN connects automatically at VM boot.
  • Incredibuild sees all build agents and distributes builds across machines.
  • No manual network configuration required.

4️⃣ Compilation Workflow

  • Edit code locally in VS2022 or via VS Code Online.
  • Push changes to Git; VM automatically triggers headless builds using MSBuild or UnrealBuildTool.
  • Build logs and optional notifications (webhooks) are captured.
  • No manual clicks needed — fully automated workflow.

5️⃣ Optional Enhancements

  • Multiple VM profiles for different projects using the same base image.
  • Auto-update scripts for Unreal Engine, VS, and dependencies.
  • Shortcuts to open latest projects immediately.
  • Automated rollback to snapshots if builds fail.

Azure Community Platform & Cloud VM Management

The platform is free, open-source, and community-driven. It allows developers to manage Windows and Ubuntu VMs with pre-configured development tools and automated workflows.

Included Features

  • Win10 Remote: GPU Windows 10 computers for development.
  • Forgejo: Self-hosted Git alternative.
  • VSCode Server: Browser-based code editor.
  • MailCow, Plane, N8N, SearNGX, Zammad, Paperless, Answer, ByteStash: pre-configured apps for full development workflow.

Automated VM Workflow

  1. Provide credentials for GitHub & Azure.
  2. Automatic setup of Azure Function Apps, GitHub Actions workflows, and environment variables.
  3. Template sync to pre-configured applications.
  4. Platform ready for production with automated deployment.

Cloud Windows Image Builder

This tool allows creating, customizing, and exporting Windows VMs as bootable VHDs.

Benefits

  • Rapidly build customized Windows images with all creative software installed.
  • Export images online or offline.
  • Save time compared to manual setup in Azure portal.

Workflow

  1. Create VM using create_vm_windows or create_vm_ubuntu.
  2. Install software and configure environment.
  3. Snapshot VM to save state.
  4. Convert VM into Hyper-V fixed-size VHD using create_vm_hyperv.
  5. Export VHD for USB or cloud deployment.

API Endpoints

  • /api/create_vm_windows
  • /api/create_vm_ubuntu
  • /api/create_vm_hyperv
  • /api/snapshot_vm
  • /api/clone_vm
  • /api/delete_vm
  • /api/list_vm

Summary

This platform and workflow let you:

  • Spin up disposable development VMs instantly.
  • Sync code automatically from Git repositories.
  • Connect to OpenVPN and use Incredibuild for distributed builds.
  • Collaborate with others using VS Code Online while retaining full local compilation.
  • Export fully-configured Windows images for offline or cloud use.

Essentially, it’s a fully automated, collaborative, and reusable development environment for complex projects like Unreal Engine, reducing setup time from hours to minutes.
